# 🗺️ Roadmap — Engenheiro de Software Back-End

> Da base ao topo: hardware, sistemas operacionais, Python, redes e banco de dados.  
> Estudo estruturado com foco em entender **por que** as coisas funcionam, não só **como** usá-las.

---

## Sobre este projeto

Este repositório documenta minha jornada de transição para desenvolvimento back-end.

Não é um roadmap genérico da internet. É o mapa real que recebi do meu gestor técnico quando perguntei o que precisava saber para me tornar desenvolvedor — e decidi transformar isso em estudo público.

**Trajetória:** Exército Brasileiro (8 anos) → Suporte Técnico → Implementação de Software → Dev Back-End

---

## Método de Estudo

1. **Assistir** — Playlist "Programming Concepts" do canal Akitando, na ordem
2. **Anotar** — Caderno físico com o que não sabia e o que precisa aprofundar
3. **Pesquisar** — Ir fundo em cada conceito anotado
4. **Replicar** — Escrever código Python que demonstra o conceito
5. **Publicar** — Transformar o aprendizado em post no LinkedIn
